Git is a powerful Source Code Management tool.

One of the most notable difficulties is that of losing work when porting the Unixy tool to Windows. 
Another issue is the comparatively slow uploading and downloading of files.
 
Msysgit is one the best packages for compiling GIT for windows. 
Msysgit is a user friendly tool and is very easy to install compared to its alternatives
